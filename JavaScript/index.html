JavaScript

Author: liaozehao Date: 十一月 17, 2020  16:27:20

使用方式

  • 在HTML中，JavaScript代码必须位于<script>与</script>标签之间。

  • JavaScript函数是一种JavaScript代码块，它可以在调用时被执行。

  • <head>或<body>中都能放置JavaScript。

  • 把脚本置于<body>底部，可改善显示速度。

显示方案

  • 使用 window.alert() 写入警告框；
  • 使用 document.write() 写入HTML输出；
  • 使用 innerHTML 写入HTML元素；
  • 使用 console.log() 写入浏览器控制台。
使用innerHTML

  • 使用id定义HTML元素

  • document.write() 如果在HTML文档完全加载后使用将会删除所有的已有的HTML，只用于测试；

使用window.alert()

  • 使用警告框来显示数据

使用console.log()

  • 使用控制台来显示数据

JavaScript语句

  • 在HTML中，JavaScript语句是由web浏览器“执行”的“指令”。
  • JavaScript程序常被称为JavaScript代码。
  • 每条可执行语句后要加英文分号。
JavaScript关键词
关键词 描述
break 终止switch或循环
continue 跳出循环并在顶端开始
debugger 停止执行JavaScript，并调用调试函数（如果可用）
do … while 执行语句块，并在条件为真的时候重复代码块
for 标记需被执行的语句块，只要条件为真
function 声明函数
if … else 标记需被执行的语句块，根据某个条件
return 退出函数
switch 标记需被执行的语句块，根据不同的情况
try … catch 对语句块实现错误处理
var 声明变量

JavaScript变量

  • JavaScript变量是存储数据值的容器。

  • 所有Java变量必须以唯一的名称的标识，这些唯一的名称为标识符

    • 名称可包含字母、数字、下划线和美元符号
    • 名称最好以字母开头，不可以数字开头
    • 大小写敏感
    • 保留字无法作变量名

  • = 是赋值运算符，**==** 是等于

  • 通过var声明变量值，只是声明的话，变量没有值（技术上它的值是undefined）

  • 字符串也能相加，但字符串将会被级联

  • var x = 3 + 5 + "8";  //输出x=88
var x = "8" + 3 + 5;  //输出x=835

  • 当JavaScript到达 return 语句，函数将停止执行，并返回值，或者返回执行某条代码

  • name 假设其是一个函数名，name 引用的是函数对象，name() 引用的是函数结果

  • 函数能直接当作变量使用

  • 在函数中声明的变量会成为局部变量，局部变量在函数开始时创建，在函数完成时删除

对象

  • 对象也是变量，但对象包含多个值

  • javascript对象是被命名值的容器

  • 访问对象属性的方式

  • 对象方法是在对象上执行的动作，方法是作为属性来存储的函数

  • 通过 objectName.methodName() 访问对象方法

  • JavaScript对象无法进行对比，返回值始终为“false”

  • 避免字符串、数值或逻辑对象

JavaScript事件

  • 常见的HTML事件

    事件 描述
    onchange HTML元素已被改变
    onclick 用户点击了HTML元素
    onmouseover 用户把鼠标移动到HTML元素上
    onmouseout 用户把鼠标移开HTML元素
    onkeydown 用户按下键盘按键
    onload 浏览器已经完成页面加载

字符串

  • length 可返回字符串的长度

  • indexOf() 方法返回字符串中指定文本首次出现的位置

  • lastIndexOf() 方法返回指定文本在字符中最后一次出现的位置

  • search() 方法搜索特定值的字符串，并返回匹配的位置

  • slice() 提取字符串的某个部分并在新字符串中返回被提取的部分

  • substring() 类似于 slice() ，但 substring() 无法接受负值

  • substr() 类似于 slice() ，但 substr() 第二个值是用来指定提取部分长度，如果省略第二个参数，将会提取剩余部分，其第二个参数为负

  • indexOf() 与 search() 区别

    • search()方法无法设置第二个开始位置参数
    • indexOf()方法无法设置更强大的搜索值（正则表达式）

  • 转义字符：\ **，\** 能让你在字符串中换行，但某些浏览器不支持

  • 当使用 new 相等运算符时，相等字符串是相等的

  • replace() 方法用另一个值替换在字符串中指定的值

  • toLowerCase() 把字符串转换为小写，toUpperCase() 把字符串转换为大写

  • concat() 连接两个及以上字符串

  • trim() 方法删除字符串两端的空白符

  • charAt() 方法返回字符串中指定下标（位置）的字符串

  • charCodeAt() 方法返回字符串中指定索引的字符unicode编码

  • split() 将字符转换为数组

数字

  • JavaScript数值始终是64位的浮点数

  • 精度

    • 整数（不使用指数或科学计数法）会被精确到15位

    • 小数的最大数是17位，但浮点的算数并不总是100%精准，使用乘除法有助于解决上面的问题

  • 数字与字符串相加将会是一个字符串

  • 数字字符串能与数字或数字字符串进行除加以外的其他数字运算

  • NaN 属于JavaScript保留词，指示某个数不是合法数
